The Password Reset application allows you to reset your password. Enter your user name in the User Name field. Select the Continue button to proceed with the password reset process. On the next page, you are shown your password hint and you must specify the reply. The Reset Password button then performs the password reset operation.The new password will be valid but expired, allowing you to reset your password on the next login.
